## 🆘 SUPPORT

### If Something Doesn't Work
1. Open DevTools (F12)
2. Check Console tab for errors
3. Check Application > localStorage for data
4. Clear cache: Ctrl+Shift+Delete
5. Refresh page
6. Try incognito mode

### Quick Fixes
- Can't redirect: Check browser console for errors
- Data missing: Check localStorage in DevTools
- Pending not showing: Verify appointment form ID is 'appointmentForm'
- Admin can't approve: Confirm logged in with admin/admin123
