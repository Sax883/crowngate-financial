# Quick Start Guide - Crowngate Financial Group Website

## 🚀 Getting Started (5 Minutes)

### Step 1: Open the Website
1. Navigate to the project folder: `crowngate financialgroup`
2. **Double-click** `start.html` to open in your browser
3. You'll see a welcome hub with all available links

### Step 2: Explore Client Features
- **Click "Main Website"** to see the professional homepage
- **Click "Create Account"** to register as a new client
- **Click "Book Appointment"** to schedule a consultation
- **Click "Contact Support"** to see contact options
- **Use the live chat widget** (bottom right) to test customer support

### Step 3: Test Admin Dashboard
1. Click **"Admin Login"** from the start page
2. Use demo credentials:
   - **Username:** `admin`
   - **Password:** `admin123`
3. Explore the admin features:
   - View dashboard statistics
   - Check customer messages
   - Manage appointments
   - Review sign-ups
   - Manage clients
   - Configure settings

## 🐛 Troubleshooting

### Data Not Saving?
- Check if localStorage is enabled
- Try a different browser
- Clear cache and cookies
- Check browser console for errors

### Page Not Loading?
- Ensure all files are in correct folders
- Check file paths in HTML files
- Use a local server instead of opening files directly

### Chat Widget Not Working?
- Ensure JavaScript is enabled
- Check browser console for errors
- Verify main.js is loaded

### Admin Dashboard Not Loading?
- Clear browser cache
- Log in again with correct credentials
- Check admin/admin.js is loaded
